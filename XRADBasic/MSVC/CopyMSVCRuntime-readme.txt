Runtime-библиотеки MSVC 2015
----------------------------

Рантайм-библиотеки состоят из двух частей:
- MSVC Runtime, аналогичный используемому в предыдущих версиях MSVC
(msvcp140.dll, vcruntime140.dll; vcomp140.dll — для OMP);
- Universal CRT (UCRT) — нововведение MSVC 2015, также используется в MSVC 2017,
в Windows 10 поставляется и обновляется в составе операционной системы.

Библиотеки могут устанавливаться в операционной системе с помощью модулей
установки VCRedist, а могут распространяться с приложением и располагаться в
директории приложения.

Ниже приведено расположение redist-файлов библиотек, предназначенных для
распространения с приложением, а также файлов отладочных версий библиотек,
не предназначенных для распространения с приложением, но необходимых для запуска
отладочных версий приложения.



MSVC Runtime
------------

Расположение redist-файлов рантайм-библиотек MSVC 2015 в системе:
"C:\Program Files (x86)\Microsoft Visual Studio 14.0\VC\redist\"

Дополнительные файлы отладочной версии рантайма (не redist!) расположены здесь:
"C:\Program Files (x86)\Microsoft Visual Studio 14.0\VC\redist\debug_nonredist\"



UCRT
----

В текущей версии мы используем комплект файлов из Windows 10 SDK 10.0.10586.212.
Версии файлов: 10.0.10586.212. Эта версия SDK соответствует MSVC 2015.

Расположение redist-файлов UCRT в системе:
"C:\Program Files (x86)\Windows Kits\10\Redist\ucrt\DLLs\"

Версии файлов redist зависят от установленного пакета Windows 10 SDK.
При установке более нового пакета (например, при установке MSVC 2017)
предыдущие версии файлов не сохраняются.

Дополнительные файлы отладочной версии UCRT (не redist!) расположены здесь:
"C:\Program Files (x86)\Microsoft SDKs\Windows Kits\10\ExtensionSDKs\Microsoft.UniversalCRT.Debug\10.0.10586.0\Redist\Debug\"
